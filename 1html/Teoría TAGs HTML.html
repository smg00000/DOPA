Introducción a HTML

¿Qué es HTML?

  • HTML (Lenguaje de Marcas de Hipertexto) es la base para dotar de contenido a una página web.
  • Utiliza un sistema de marcas (etiquetas) para definir el formato, las partes del documento y las referencias.

Importancia de los Estándares

  • Seguir los estándares de HTML asegura que los navegadores interpreten y muestren la información correctamente.

Estructura Básica de un Documento HTML

Toda página web comienza con esta estructura esencial:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Mi Página</title>
</head>
<body>
  <h1>¡Hola, mundo!</h1>
</body>
</html>
4 / 20

La Cabecera (<head>)

Contiene metadatos esenciales:

  • <title>: Título en la pestaña del navegador.
  • <meta charset="UTF-8">: Codificación de caracteres.
  • <meta name="description">: Descripción para SEO.
  • <link rel="stylesheet">: Enlaza CSS.
Ejemplo práctico: 
<head>
  <meta charset="UTF-8">
  <title>Blog de Tecnología</title>
  <meta name="description" content="Artículos sobre desarrollo web">
  <link rel="stylesheet" href="estilos.css">
</head>
5 / 20

El Cuerpo (<body>)

Aquí va todo el contenido visible:

  • Texto, imágenes, enlaces, formularios, etc.
  • Organizado con etiquetas semánticas como <header>, <main>, <footer>.
Ejemplo de contenido: 
<body>
  <header>
    <h1>Mi Sitio Web</h1>
  </header>
  <main>
    <p>Este es el contenido principal.</p>
  </main>
  <footer>
    <p>© 2024</p>
  </footer>
</body>

Formato de Texto

Texto normal, texto en negrita (strong), texto en cursiva (em), texto resaltado (mark), negrita sin énfasis (b), cursiva sin énfasis (i).

Código HTML: 
<p>Texto normal, <strong>texto en negrita</strong>,
<em>texto en cursiva</em>, <mark>texto resaltado</mark>,
<b>negrita sin énfasis</b>, <i>cursiva sin énfasis</i>.</p>

Divs y Contenedores

El elemento <div> es un contenedor genérico para agrupar elementos:

Sección 1

Contenido dentro de un div.

Sección 2

Otro bloque de contenido.

Código HTML: 
<div>
  <h3>Sección 1</h3>
  <p>Contenido...</p>
</div>
<div>
  <h3>Sección 2</h3>
  <p>Otro bloque...</p>
</div>

Enlaces Internos

Metadatos Avanzados

Ejemplos de metadatos útiles en el <head>:

<!-- Vista móvil -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<!-- Descripción para SEO -->
<meta name="description" content="Aprende HTML desde cero">

<!-- Palabras clave (menos relevante hoy) -->
<meta name="keywords" content="HTML, web, desarrollo">

<!-- Autor -->
<meta name="author" content="Tu Nombre">

<!-- Icono de la página -->
<link rel="icon" href="favicon.ico">

Resumen y Conclusiones

  • HTML es el esqueleto de la web, proporcionando la estructura fundamental de cualquier página.
  • La separación entre <head> (metadatos) y <body> (contenido) es un concepto central.
  • HTML ofrece una gran variedad de etiquetas para dar formato al texto, crear listas, enlaces, incrustar contenido multimedia, organizar datos en tablas y recoger información mediante formularios.
  • Siempre prioriza la semántica y el uso correcto de las etiquetas para mejorar accesibilidad y SEO.
